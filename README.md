## 🚀 Projetos em Destaque

### 🟢 [Mola Mestre - Gestão e Educação Financeira](https://github.com/CaioHGuedes/Mola-Mestre)

Uma plataforma all-in-one para investidores que combina o gerenciamento de carteira em tempo real com um ecossistema de aprendizado gamificado e atualizações do mercado financeiro.

**Tecnologias:**

- Frontend: Next.js, TypeScript, Tailwind CSS, Shadcn UI, Recharts, Framer Motion
- Backend: Next.js API Routes, MongoDB (Mongoose), TanStack Query
- Features: Cotações da B3 ao vivo, Feed de Notícias, Quiz Educacional Interativo, Gestão de Metas e Transações.

**Destaques:**

- Consumo robusto da API Brapi para fornecer cotações e variações de ativos da B3 atualizadas automaticamente.
- Módulo de Quiz interativo que testa e reforça conhecimentos sobre investimentos.
- Uso estrito de TypeScript com interfaces compartilhadas (DTOs) entre Frontend e Backend.
- Interface responsiva com micro-interações (Framer Motion) e visualização de dados complexos simplificada (Recharts).
